Angular Selector Queries Test Page

This is an angular directive based on https://github.com/ahume/selector-queries.

The two menus below both have identical HTML but are presented differently depending on their current width. At a 'typical' browser width and font size the menu in the smaller column simply lists the links. In the larger column it displays extra images and lays out the options in a grid.

Moreover, these two menus are wrapped by an element that receives a class when the viewport is below 480px, causing its width to go 100%.

This is achieved with a small piece of JavaScript, which is activated by the code shown below.

<ul class="my-list" ng-sq ng-sq-min-width="30em" ng-sq-class="wide">

.. and the classic style is available:

<ul class="my-list" ng-sq="min-width:30em=wide">

Adding the ng-sq attribute above causes a class value of wide to be added to the element when it is wider than 30 ems, and extra CSS is applied to the element based on that class value.

Currently you can use em or px units.

NOTE: At this moment, multiple parameters are supported only via the classic notation.
